# 剧本获取策略

> 行业 0.16% 的爆款率说明"做什么"比"怎么做"重要得多。剧本质量是短剧成败的第一因素。本文整理获取优质剧本的多种路径，以及面向海外市场的本地化换皮方法论。

## 七种剧本获取路径

### 1. 小说改编 — 最成熟的路径

从已有的好小说中提炼故事，改编为短剧节奏。

- 优势：故事经过读者验证，人物和冲突已成型
- 风险：版权成本、小说节奏和短剧节奏差异大（需要大量删减和重构）
- 关键操作：不是"忠实改编"，而是提取核心冲突和人设，用短剧节奏重写

### 2. 爆款换皮 — 最高效的路径

把已验证的爆款内容换一种世界观或模式。

- 优势：故事结构和情绪节奏已被市场验证
- 风险：同质化严重、观众审美疲劳
- 关键操作：换的不只是表面设定，而是要找到新的"信息差"——让观众觉得新鲜但又熟悉

### 3. 平台数据倒推需求 — 需求侧驱动

不从已有内容出发，而是从"观众正在找什么"出发。

- 数据来源：平台搜索热词、评论区高频诉求（"求续集"、"想看 XX 题材"）、爆款弹幕中的"意难平"（观众想看但没被满足的情节走向）
- 品类空白分析：所有人都在做玄幻时，都市悬疑可能是蓝海
- 优势：直接对准未被满足的需求
- 适合海外市场：分析 TikTok、YouTube Shorts、ReelShort 等平台的评论和搜索趋势

### 4. 提炼情绪公式 — 比换皮更深一层

不复制剧情，复制**情绪曲线**。

```
一部爆款的情绪节奏：
  屈辱 → 隐忍 → 觉醒 → 碾压 → 反转

这个情绪弧可以套到任何世界观：
  古代宫廷：受辱妃子 → 忍辱负重 → 掌握权力 → 碾压对手
  现代都市：被开除员工 → 默默创业 → 公司上市 → 收购前东家
  西方奇幻：被驱逐骑士 → 流浪修炼 → 屠龙归来 → 夺回王位
```

和第 2 种（换皮）的区别：换皮是复制具体故事结构，情绪公式是提炼更底层的人类心理需求（复仇、逆袭、证明自己），然后用全新的情节去满足这个需求。

与 short-maker 的关系：这和 Strategy Engine 的思路一致——策略不只是镜头语言层面的，还可以上升到叙事情绪层面。情绪公式可以成为策略库的一部分。

### 5. 钩子先行，反向生长 — 成本最低的验证方式

不是先写完整故事再找钩子，而是**先设计钩子，再围绕钩子长出故事**。

短剧的前 3 秒决定生死：
1. 先生成大量高冲击力的开头概念（"醒来发现自己是皇帝的私生子"）
2. 用封面 + 标题做小范围投放测试点击率
3. 只对 CTR 高的概念继续生成完整剧本

测试阶段只需要一张封面图和一句话，不需要生成整部剧。极低成本筛选出有潜力的概念。

### 6. 跨媒介挖矿 — 已验证但未被短剧化的素材

不只是小说，很多有故事验证但没被短剧化的来源：

- 知乎高赞故事/经历贴（天然有"钩子+反转"结构）
- 微博/小红书的情感热帖
- 民间传说/神话的现代改编（《西游玉帝》就是这个路径，2.3 亿播放量）
- 热门游戏的世界观和剧情线
- Reddit 的 r/AITA、r/tifu、r/relationship_advice（海外市场的天然素材库）

共同点：**已经被受众验证过有吸引力，但还没被短剧形式消费过**。

### 7. 批量生成 + 概念验证漏斗 — 最适合自动化

用 short-maker 的自动化能力在漏斗前端做大量试错，只对验证过的概念投入完整生产资源：

```
LLM 批量生成 200 个剧本概念（一句话梗概）
  → 规则过滤：题材重复度、关键词热度 → 留 30 个
  → 为 30 个生成封面 + 标题
  → 小范围投放测试 CTR → 留 5 个
  → 5 个生成完整第 1 集
  → 投放测试完播率 → 留 2 个
  → 2 个继续生产完整剧集
```

和互联网产品的 MVP 验证是同一个逻辑。关键在于**每一层的成本足够低**，才能支撑大量试错。

## 海外换皮方法论 — 中国爆款 → 海外市场

### 为什么中国短剧适合出海

中国短剧行业在故事结构、节奏设计、钩子密度上积累了大量经验（FreeToon 40% 爆款率 vs 行业 0.16%）。这些 know-how 是可迁移的——**人类的基础情感需求跨文化通用**（复仇、逆袭、爱情、权力），但表达方式需要本地化。

ReelShort（中文在线旗下）已经验证了这条路径：用中国短剧的叙事 know-how + 美国演员/场景拍摄，2024 年营收超 3 亿美元。

### 换皮的三个层次

#### 层次 1：表面换皮（最低成本，最高风险）

只换语言和画面风格，故事结构照搬。

```
中国版：霸道总裁爱上灰姑娘，豪门恩怨
美国版：Rich CEO falls for small-town girl, family drama
印度版：Wealthy businessman falls for village girl, family honor
```

问题：文化隔阂会导致"怪异感"。中国式的霸道总裁行为在美国语境下可能被视为 harassment；印度观众可能需要更强的家族/种姓冲突元素。

#### 层次 2：文化适配（推荐的做法）

保留情绪公式和节奏结构，替换文化载体。

**需要替换的文化要素：**

| 维度 | 要做什么 | 示例 |
|------|---------|------|
| **权力结构** | 换成目标文化中的权力象征 | 中国：皇帝/家族族长 → 美国：科技巨头CEO/政治家族 → 印度：种姓上层/宝莱坞巨星 |
| **冲突来源** | 换成目标文化中的核心矛盾 | 中国：家族荣辱、门当户对 → 美国：阶层跨越、种族议题、个人 vs 体制 → 印度：种姓、家族安排婚姻、传统 vs 现代 |
| **逆袭路径** | 换成目标文化认可的成功方式 | 中国：修炼/权谋/商战 → 美国：创业/法律/体育 → 印度：教育/科技/宝莱坞 |
| **关系模式** | 换成目标文化的社交规范 | 中国：长辈权威不可挑战 → 美国：个人意志优先 → 印度：家庭义务优先但有代际冲突 |
| **幽默方式** | 换成目标文化的笑点 | 中国：谐音梗、身份反差 → 美国：讽刺、自嘲 → 印度：家庭闹剧、语言混搭 |
| **视觉符号** | 换成目标文化的审美 | 中国：仙侠/古装 → 美国：超英/科幻/当代 → 印度：神话/色彩浓烈/歌舞 |
| **道德底线** | 适配目标文化的价值观红线 | 中国：忠孝 → 美国：个人自由/政治正确 → 印度：家庭荣誉/宗教敏感 |

**不需要换的（跨文化通用）：**

- 情绪弧线：屈辱→逆袭→碾压 全球通用
- 节奏结构：前 3 秒钩子、每集结尾悬念、钩子密度
- 人性母题：复仇、证明自己、保护所爱的人、获得尊重
- 短剧技巧：信息差制造悬念、反转密度、升级打怪的爽感

#### 层次 3：原生故事（最高成本，最低风险）

直接用目标市场的文化素材和叙事传统生成故事，只借鉴中国短剧的**节奏 know-how 和生产方法论**。

```
美国市场原生故事：
  素材来源：Reddit 热帖、TikTok 热门话题、Netflix 热播剧评论区
  叙事 know-how：中国短剧的钩子密度、节奏控制、反转设计
  → 故事是美国的，节奏是中国的

印度市场原生故事：
  素材来源：宝莱坞经典改编、印度神话现代演绎、Quora 印度区热帖
  叙事 know-how：同上
  → 故事是印度的，节奏是中国的
```

### 各市场的爆款题材偏好

基于 ReelShort、FlexTV、DramaBox 等出海平台的公开数据：

**美国/欧美市场：**
- 狼人/吸血鬼 + 言情（werewolf romance 是 ReelShort 上最大品类）
- 豪门复仇/身份反转
- 黑手党/地下世界 + 爱情
- 超自然 + 悬疑

**印度/南亚市场：**
- 种姓跨越 + 爱情
- 家族安排婚姻 vs 自由恋爱
- 神话/宗教故事现代改编
- 贫富差距 + 逆袭

**东南亚市场：**
- 甜宠 + 校园
- 豪门 + 灰姑娘
- 穿越/重生
- 复仇 + 商战

**中东市场：**
- 家族荣誉 + 爱情
- 社会阶层 + 逆袭
- 注意宗教敏感内容的红线

### 换皮的自动化可能性

对 short-maker 而言，换皮流程可以部分自动化：

```
输入：
  - 一部中国爆款的故事蓝图（StoryBlueprint）
  - 目标市场（us/india/sea/mena）
  - 目标风格（live_action/manga/3d）

自动化的部分：
  - 文化要素替换表（权力结构、冲突来源、逆袭路径...按市场预设）
  - 情绪曲线保留，情节重新生成
  - 角色名、场景、视觉符号自动适配
  - 对话风格和幽默方式适配

需要人工的部分：
  - 文化敏感性审核（避免冒犯）
  - 当地市场趋势判断
  - 最终品质把关
```

## 与进化机制的关系

剧本获取策略和 [evolution-loop.md](evolution-loop.md) 中的进化机制形成上下游关系：

```
剧本获取（本文）          → 决定"做什么"
Pipeline 生产              → 决定"怎么做"
投放 + 数据反馈（进化机制）→ 验证"做对了没"
                              ↓
                          反馈回剧本获取策略
```

具体来说：
- 概念验证漏斗（路径 7）的投放数据直接喂入进化循环
- 各市场的题材偏好不是静态的，需要持续用投放数据验证和更新
- 情绪公式库（路径 4）应该和策略库一样，积累使用次数和效果数据
- 换皮的文化适配规则也应该由数据驱动——哪些替换有效、哪些无效，需要积累
